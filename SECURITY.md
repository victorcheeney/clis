# Security Policy

## Supported Versions

Security fixes are applied on `main`.

## Reporting a Vulnerability

Please do not open a public GitHub issue for exploitable vulnerabilities.

Preferred paths:

1. Use GitHub's private vulnerability reporting for this repository, if enabled.
2. If private reporting is unavailable, contact the repository owner through GitHub: https://github.com/victorcheeney

Include:

- affected route or file
- reproduction steps
- impact
- any suggested fix

You will get an acknowledgment as soon as the report is reviewed.
